<h1 align="center">🛍️ Customer Shopping Behavior Analysis</h1>

<p align="center">
An end-to-end Data Analytics project built using <strong>SQL, Python, and Power BI</strong> to analyze customer shopping behavior, purchasing trends, sales performance, and operational insights through interactive dashboards.
</p>

<hr>

<h2>📌 Project Overview</h2>

<p>
This project analyzes customer shopping behavior using a structured data analytics workflow. The dataset was first cleaned and prepared using Python, then stored and analyzed using SQL to answer business questions. Finally, an interactive Power BI dashboard was created to visualize key insights and support data-driven decision-making.
</p>

<p>
The dashboard helps understand customer purchasing patterns, product preferences, payment methods, shipping choices, seasonal trends, and customer loyalty through easy-to-understand visualizations and KPIs.
</p>

<hr>

<h2>🎯 Project Objectives</h2>

<ul>
    <li>Understand customer purchasing behavior.</li>
    <li>Analyze sales performance across different product categories.</li>
    <li>Identify loyal, repeat, and new customers.</li>
    <li>Evaluate payment methods and shipping preferences.</li>
    <li>Track seasonal sales trends and customer ratings.</li>
    <li>Create an interactive dashboard for business reporting.</li>
</ul>

<hr>

<h2>⚙️ Project Workflow</h2>

<ol>
    <li>Collected and explored the shopping dataset.</li>
    <li>Cleaned and preprocessed the data using Python.</li>
    <li>Stored and queried the data using SQL.</li>
    <li>Performed business analysis using SQL queries.</li>
    <li>Created KPIs and DAX measures in Power BI.</li>
    <li>Designed an interactive dashboard with multiple report pages.</li>
</ol>
